# NOVA — AI 기반 차세대 플랫폼

NOVA는 AI 기술과 미래지향적 사용자 경험을 결합한 인터랙티브 웹 플랫폼 프로젝트입니다.

사용자가 단순히 정보를 읽는 것을 넘어, 애니메이션과 동적인 UI를 통해 차세대 AI 플랫폼의 비전과 가치를 경험할 수 있도록 설계했습니다.

## 프로젝트 소개

NOVA는 AI 기반 디지털 플랫폼을 주제로 제작한 React 프로젝트입니다.

미래지향적인 디자인과 인터랙티브 요소를 활용하여 사용자에게 몰입감 있는 웹 경험을 제공하는 것을 목표로 개발했습니다.

### 주요 목표

- React 기반 컴포넌트 설계
- 인터랙티브 UI 구현
- 사용자 중심 UX 설계
- 모던 웹 개발 패턴 적용
- 반응형 웹 구현

---

## 주요 기능

### Hero Section

- 플랫폼 핵심 메시지 전달
- CTA(Call To Action) 버튼 제공
- 시각적 임팩트를 위한 대형 타이포그래피 구성

### AI Feature Showcase

- AI 자동화 기능 소개
- 카드 기반 UI 구성
- 사용자 친화적인 정보 전달

### Statistics Dashboard

- 실시간 데이터 기반 플랫폼 이미지를 표현하는 통계 섹션
- 수치 정보를 활용한 신뢰도 강화

### Interactive Experience

- 스크롤 기반 인터랙션
- Hover 애니메이션
- 부드러운 UI 전환 효과

### Contact Section

- 사용자 문의 및 서비스 연결 기능 제공
